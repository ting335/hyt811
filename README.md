# 夏培肃人物专题网站

本项目为新生实践课程结课作业，以“中国计算机之母”夏培肃院士为主题，使用 HTML 和 CSS 制作了一个多页面静态人物介绍网站。网站从基本信息、主要成就、人生经历和人物评价等方面展示夏培肃院士对中国计算机事业发展的重要贡献。

## 页面内容

- **首页**：展示网站主题、人物照片、主要身份与导航入口；
- **基本信息**：使用表格整理人物生平信息；
- **主要成就**：按时间梳理科研、教育和人才培养成果；
- **人生经历**：介绍求学、科研和推动中国计算机事业发展的历程；
- **人物评价**：展示相关机构和研究人员对夏培肃院士的评价。

## 技术实现

- HTML5 页面结构；
- CSS 内联样式设计；
- Flexbox 页面布局；
- 表格信息展示；
- 多页面相对路径导航；
- 本地图片资源引用；
- 鼠标悬停链接样式。

## 本地运行

本项目不需要安装额外依赖。下载或克隆仓库后，可以直接使用浏览器打开 `index.html`。

也可以在项目目录启动一个本地静态服务器：

```bash
python -m http.server 8000
```

随后访问：

```text
http://localhost:8000
```

## 部署到 GitHub Pages

1. 将项目文件上传到 GitHub 仓库；
2. 打开仓库的 `Settings`；
3. 进入 `Pages`；
4. 在 `Build and deployment` 中选择从分支部署；
5. 选择 `main` 分支和根目录 `/root`；
6. 保存并等待 GitHub 生成访问地址。

## 课程收获

通过本项目，完成了从内容规划、页面结构设计到样式调整和多页面组织的完整静态网站开发过程，实践了 HTML 基础标签、CSS 布局、图片引用、表格设计和页面跳转等前端基础知识。

## 已知不足

- 页面样式主要写在各 HTML 文件内部，存在重复代码；
- 部分尺寸使用固定像素值，在手机和小尺寸屏幕上的适配效果有限；
- 个别页面存在不规范或重复的 HTML 结束标签；
- 导航栏尚未抽取为统一组件；
- 当前未使用 JavaScript 实现交互功能。

后续可以将公共样式拆分为独立的 `style.css`，补充响应式布局和统一导航，并进一步改善语义化标签与可访问性。
